# Wheal

Essentially reinventing popular tools/programs as an exercise for learning new languages. A few languages I'd like to explore are:

- Rust
- Elixir
- Go
- Zig
- OCaml
- Assembly

This repository is just going to be a hodge-podge of a bunch of languages and a bunch of tools without any direction.
Like I said, more of a learning exercise than anything else.

## Logs

Each project will contain a LOG.md file that notes certain things such as:

- Language used
- Start/end dates
- Proficiency in lanaguage
- Project description

## Future

Here is a small list of programs I think would be cool to reinvent:

- tmux
- vi
- git
- jrnl

The list will grow and shrink as I think of new ideas and check off others as completed.

## AI Usage

There is no AI used in any writing of code, only in research. Again, this project is meant to be for learning purposes
and generating code with AI destroys that purpose.
